Methodology

Data Research

Transforming Data

  • Data loaded to Mongo databases from an api or json file were cleaned before insertion into a collection
  • Json file data was verfied before use in a chart or table
  • CSV file data was cleaned and verified before use in a chart

Data Repositories

Data Exploration

  • Dynamic bubble charts used to show NASDAQ and NYSE activity from the first two quarters of 2021
