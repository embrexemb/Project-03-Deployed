About the results

Methodology

Data Research

  • Data downloaded from epa.org
    Areas selected: Raleigh, NY, LA, Cheyenne, El Paso, Seattle Years: 2016, 2018, 2020
  • Data sets from Kaggle and UCI were studied and used as a guide for collecting and collating air quality Pollutants
    See urls - listed here

Transforming Data

  • Daily, monthly and annual monitoring data was collected fron the EPA data portal
  • All monitoring data files were sorted, filtered and cleaned to remove blanks. Jupyter Notebooks and Excel macros used to filter and sort data
  • All data points came from monitoring data
  • Json file data was verfied before use in a chart or table

Data and APIs in use

  • Machine Learning models and results saved to Notebooks folder with datasets used in training and prediction sets
  • Jupyter Notebooks files used for cleaning data.
  • CSV files used as data inputs for Website visualizations and Machine Learing models
  • Javascript files used for dynamic chart display
  • The website api is deployed to Heroku at http:############## with a flask web service.

Data Visualizations

  • Line Chart used to show top pollutants distribution.
  • Scatter chart used to show pollution distribution accross cities.
  • Donut chart used to show pollutants distribution in cities vs rural.

Predictive Modeling

  • The best results came from a Gaussian process regression (gpr)
  • 18 data sets were set up to test for the top 3 Pollutants that affect Air Quality for each city
  • See Machine Learning Overview for results and details
